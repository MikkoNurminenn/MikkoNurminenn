# Mikko Nurminen

I build data-driven tools and product-style web experiences.

My work sits between analysis and product development: Python notebooks, applied machine learning, web interfaces, and workflow tools that make complex information easier to use.

Right now I am focused on projects that mix analytical thinking with practical user-facing execution, especially in sports, consumer web, and small business contexts.

## Featured Work

- [Creme Crust](https://github.com/MikkoNurminenn/creme-crust) - bilingual Next.js brand site for a pastry business, built around preorder storytelling, strong visuals, and Instagram-aware content.
- [GDP vs Life Expectancy](https://github.com/MikkoNurminenn/GDP_vs_LifeExpectancy) - Python analysis of GDP and public-health trends across six countries between 2000 and 2015.
- [Health Insurance Analysis](https://github.com/MikkoNurminenn/HealthInsuranceAnalysis) - notebook-based study of U.S. medical insurance costs using exploratory analysis and predictive modeling.
- [WordPress Author Site](https://github.com/MikkoNurminenn/wordpress-author-site) - case study of a lightweight author website built with native WordPress capabilities instead of a plugin-heavy stack.
